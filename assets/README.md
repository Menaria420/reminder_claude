# Assets Directory

Place your app assets here:

## Required Files:
- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1284x2778 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Recommended Icon Guidelines:

### App Icon (icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Should work well when scaled down

### Splash Screen (splash.png)
- Size: 1284x2778 pixels (or larger)
- Format: PNG
- Design with safe zones in mind

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Will be masked on Android

You can generate these using:
1. [Expo Icon Builder](https://buildicon.netlify.app/)
2. [App Icon Generator](https://appicon.co/)
3. [Figma](https://www.figma.com/) - Design your own

## Color Palette for Icons:
- Primary: #667EEA
- Secondary: #764BA2
- Success: #10B981
- Warning: #F59E0B
- Danger: #EF4444
